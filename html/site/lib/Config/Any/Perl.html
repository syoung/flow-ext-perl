NAME

Config::Any::Perl - Load Perl config files

DESCRIPTION

Loads Perl files. Example:

    {
        name => 'TestApp',
        'Controller::Foo' => {
            foo => 'bar'
        },
        'Model::Baz' => {
            qux => 'xyzzy'
        }
    }

METHODS

extensions( )

return an array of valid extensions (pl, perl).

load( $file )

Attempts to load $file as a Perl file.
