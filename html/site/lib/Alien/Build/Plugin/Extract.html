NAME

Alien::Build::Plugin::Extract - Extract Alien::Build plugins

VERSION

version 2.21

SYNOPSIS

 use alienfile
 share {
   plugin 'Extract' => 'tar.gz';
 };

DESCRIPTION

Extract plugins extract packages that have been downloaded from the internet. Unless you are doing something unusual you will likely want to use the Alien::Build::Plugin::Extract::Negotiate plugin to select the best Extract plugin available.

Alien::Build::Plugin::Extract::ArchiveTar
Alien::Build::Plugin::Extract::ArchiveZip
Alien::Build::Plugin::Extract::CommandLine
Alien::Build::Plugin::Extract::Directory
Alien::Build::Plugin::Extract::Negotiate

SEE ALSO

Alien::Build, Alien::Build::Plugin
