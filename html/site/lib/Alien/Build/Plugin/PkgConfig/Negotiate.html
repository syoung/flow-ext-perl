NAME

Alien::Build::Plugin::PkgConfig::Negotiate - Package configuration negotiation plugin

VERSION

version 2.21

SYNOPSIS

 use alienfile;
 plugin 'PkgConfig' => (
   pkg_name => 'libfoo',
 );

DESCRIPTION

This plugin provides Probe and Gather steps for pkg-config based packages. It picks the best PkgConfig plugin depending your platform and environment.

PROPERTIES

pkg_name

The package name.

atleast_version

The minimum required version that is acceptable version as provided by the system.

exact_version

The exact required version that is acceptable version as provided by the system.

max_version

The max required version that is acceptable version as provided by the system.

minimum_version

Alias for atleast_version for backward compatibility.

METHODS

pick

 my $name = Alien::Build::Plugijn::PkgConfig::Negotiate->pick;

Returns the name of the negotiated plugin.

ENVIRONMENT

ALIEN_BUILD_PKG_CONFIG

If set, this plugin will be used instead of the build in logic which attempts to automatically pick the best plugin.
