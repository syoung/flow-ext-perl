NAME

Net::SSH::Perl::Proxy - Use TCP proxy to connect to a host via SSH

SYNOPSIS

    my $ssh = Net::SSH::Perl::Proxy->new('myhost',
        proxy => {
            host => 'proxyhost',
            port => 1080,
        }
    );

    $ssh->login();

DESCRIPTION

This class extends Net::SSH::Perl to allow connections through a SOCKS proxy.
