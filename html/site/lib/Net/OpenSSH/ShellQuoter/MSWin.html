NAME

Net::OpenSSH::ShellQuoter::MSWin - Quoter for Win32::CreateProcess

DESCRIPTION

This quoter is intended for interaction with SSH servers running on Windows which use the Win32::CreateProcess system call to launch the requested command.

Because of Win32::CreateProcess not doing wildcard expansion, glob quoting just quotes everything.
